Applications of Computing in Weather Modelling

Data Collection

Collecting data is the first part of predicting future weather. This is gathered by:

  • Satellites
  • Weather Stations
  • Radar
  • Remote Buoys
  • Weather Balloons

These collect data such as:

  • Temperature
  • Wind speed & direction
  • Pressure
  • Humidity

Data Processing

The next step is optimising and processing the data. This is called data assimilation. It is required because of the large volumes of global data collected. Machine learning is then used to create models of weather patterns, and neural networks are used to predict weather in advance.

Using the models

Weather maps are created, and these allow for local weather to be fitted into a larger weather system. In the UK, the Met Office model weather and release forecasts for the public. These models can help to predict extreme weather events, which allow for preparation for hurricanes, tornadoes, snowstorms and more. This means that authorities can call for evacuations or put protection in place to prevent damage or loss of life. Also, accurate weather predictions can allow industries, particularly agrictulture, to prepare and put measures in place to protect their business.

Hardware and Model Refinement

Some of the fastest supercomputers in the world are used in this field, as there are very large amounts of data to process. These computers are able to run near-realistic simulations and the models are being finetuned and improved constantly. In many cases, multiple different 'expert' models are run on the same data, and all combined to create a consensus prediction of the future weather. There is a constant cycle occuring daily to constantly improve prediction:

  1. Collection of real-time data
  2. Prediction of future weather
  3. Refinement of prediction models based on actual weather
